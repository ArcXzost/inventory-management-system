# **Inventory Management System**

An advanced inventory management system designed to streamline stock operations, track orders and shipments, and provide predictive analytics using machine learning models. The application ensures efficient inventory handling, real-time insights, and user-friendly interaction.

## **Technologies Used**

- **Frontend**: Android (Kotlin)
- **Backend**: Firebase (initial setup) and Room Database
- **Charts and Graphs**: MPAndroidChart library
- **Machine Learning**: Integrated predictive models
- **Version Control**: Git

## **Scope**

- This project is only one part of a complicated network of interfaces used in the supply chain. To truly demonstrate the capabilities of this system we would require other interfaces like an e-commerce interface from which the orders are placed, a shipments interface to manage shipments bound for the warehouse. So any contributions and collaborations are welcome !!
- The prediction model used is DistilBERT as a demand forecaster. However for android usage the pyscript format is not very optimised. Contributions towards a better and optimised approach towards demand forecasting are welcome.
